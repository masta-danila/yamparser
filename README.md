# 🗺️ Парсер отзывов Яндекс Карт

Быстрый и надежный парсер отзывов с Яндекс Карт с поддержкой инкрементального обновления и автоматического решения CAPTCHA.

## ✨ Особенности

- 🚀 **Быстрое раскрытие отзывов** (4-5x ускорение)
- 🔄 **Инкрементальный парсинг** с checkpoint системой
- 🤖 **Автоматическое решение CAPTCHA**
- 📱 **Мобильная эмуляция** для обхода блокировок
- 🌐 **Быстрая система прокси** без предварительного тестирования
- 💾 **SQLite база данных** с защитой от дубликатов
- 🎯 **Фильтрация официальных ответов**
- 📊 **Детальная статистика** и отчетность

## 📋 Требования

```bash
pip install -r requirements.txt
```

## ⚙️ Конфигурация

Все настройки системы находятся в файле `config.py`:
- `SPREADSHEETS` - список таблиц Google Sheets для обработки
- `MAX_DAYS_BACK` - количество дней назад для парсинга (30 по умолчанию)
- `MAX_WORKERS` - количество потоков (20 по умолчанию)

## 🚀 Быстрый старт

### 1. Настройка прокси (опционально)

Создайте файл `proxy.txt` с вашими прокси в формате `IP:PORT:USERNAME:PASSWORD`:
```
# Формат: IP:PORT:USERNAME:PASSWORD
82.97.251.114:16172:nEVzuQ:YKAGuCukum2s
203.0.113.10:3128:user1:pass1
198.51.100.20:8080:user2:pass2

# Комментарии начинаются с #
# Пустые строки игнорируются
```

### 2. Запуск парсинга
```bash
# Парсинг по умолчанию (Advert Pro)
python get_reviews.py

# Парсинг конкретной организации
python get_reviews.py "https://yandex.ru/maps/org/organization_name/123456789/reviews/"

# Тестирование прокси
python test_proxy_integration.py
```

## 📁 Структура проекта

### Основные файлы:
- `integrated_parser.py` - Главный скрипт парсинга (Google Sheets + Яндекс.Карты)
- `reviews_parser.py` - Парсинг отзывов с Яндекс.Карт
- `selenium_captcha_solver.py` - Решение CAPTCHA

### Прокси система:
- `proxy_manager.py` - Менеджер прокси серверов
- `proxy_utils.py` - Утилиты для работы с прокси
- `test_proxy_integration.py` - Тестирование прокси
- `proxy.txt` - Файл с прокси серверами

### Вспомогательные файлы:
- `.gitignore` - Настройки git
- `README.md` - Документация проекта

## ⚙️ Настройки

В `get_reviews.py` можно изменить:

```python
# ============= НАСТРОЙКИ ПАРСИНГА =============
MAX_DAYS_BACK = 1095        # Дни для первичного парсинга (3 года)
MAX_REVIEWS_LIMIT = 2000    # Максимум отзывов за сессию
DEVICE_TYPE = "mobile"      # "mobile" или "desktop" 
WAIT_TIME = 3               # Время ожидания загрузки (сек)
# =============================================
```

## 🔄 Режимы работы

### Первичный режим  
- Парсит все отзывы за заданный период (MAX_DAYS_BACK)
- Сравнивает с данными из Google Sheets и обновляет статусы

Структура таблицы `reviews`:
```sql

- card_id (ID организации)
- author (автор отзыва)
- rating (рейтинг 1-5)
- date (дата отзыва)
- text (текст отзыва)
```
